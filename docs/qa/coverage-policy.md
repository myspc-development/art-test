# Coverage Policy

## Thresholds
- PHP ≥ 80%
- JavaScript ≥ 80%
- E2E tests cover critical flows

## Exclusions
- Generated code
- Index files

## Enforcement
Pull requests failing thresholds must be fixed before merge.
